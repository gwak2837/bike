## Requirements (Recommanded)

- macOS 13.6
- Bun 1.1
- PostgreSQL 16.3
- Docker 25.0

## Getting Started

```bash
bun i
```

#### Watch mode

```bash
bun dev
```

#### Docker mode

```bash
docker compose up -d --build --force-recreate
```

#### Bundle mode

```bash
bun run build
bun start
```

#### Compile mode

```bash
bun run compile
./app.bun
```
