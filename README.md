## Requirements (Recommanded)

- macOS 13.6
- Bun 1.1
- Node.js 20.15
- PostgreSQL 16.3

## Getting Started

```bash
bun i
```

### backend

```bash
cd backend
cp .env.template .env.development.local
bun dev
```

### frontend

```bash
cd frontend
cp .env.template .env.development.local
npm run dev
```
